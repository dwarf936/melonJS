# Particle System Example

这是一个基于melonJS引擎的粒子系统演示模块，展示了高性能的爆炸效果粒子动画。

## 功能特性

### 核心功能
1. **爆炸效果** - 实现向外扩散的动态粒子轨迹，模拟真实的爆炸物理效果
2. **交互式控制面板** - 实时调整各种粒子参数，即时预览效果变化
3. **高性能渲染** - 在1000+粒子数量下仍能保持稳定60fps帧率

### 可调参数

#### 基础设置
- **粒子总数**: 控制单次爆炸发射的粒子数量 (10-1500)
- **粒子颜色**: 自定义粒子的颜色
- **叠加模式**: 启用加法混合模式，创建更明亮的发光效果

#### 运动设置
- **初始速度**: 粒子发射的基础速度
- **速度变化**: 粒子速度的随机变化范围
- **重力**: 粒子受到的垂直重力影响
- **风力**: 粒子受到的水平风力影响

#### 生命周期
- **最小生命周期**: 粒子的最短存活时间
- **最大生命周期**: 粒子的最长存活时间

#### 缩放设置
- **初始最小缩放**: 粒子初始大小的最小值
- **初始最大缩放**: 粒子初始大小的最大值

## 使用方法

1. **点击画布**: 在游戏画布任意位置点击鼠标或触摸屏幕，即可在对应位置触发一次爆炸效果
2. **触发爆炸效果**: 点击控制面板中的"触发爆炸效果"按钮，在画布中心创建爆炸
3. **重置发射器**: 清除当前所有粒子

## 性能优化

本演示通过以下方式保证高性能:
1. 对象池复用 - 使用melonJS内置的粒子对象池，避免频繁创建和销毁对象
2. 视口剔除 - 粒子仅在视口内进行更新和渲染
3. 帧跳过机制 - 可配置跳过更新帧数以平衡性能和视觉效果
4. WebGL加速 - 利用硬件加速渲染，支持数千粒子同时运动

## 技术实现

- 使用melonJS内置的`ParticleEmitter`类管理粒子系统
- 采用React创建交互式控制面板，实现实时参数调整
- 使用TypeScript进行类型安全开发
